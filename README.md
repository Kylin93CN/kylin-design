# kylin-design
仿照antd，使用ts+react打造自己的组件库

## 安装
`npm i kylin-design -S`

## 使用

```javascript
// 引入组件
import { Button } from 'kylin-design';
// 引入组件样式
import 'kylin-design/dist/index.css';

function App() {
  return (
    <div className="App">
      <Button btnType="primary">bbbbb</Button>
    </div>
  );
}

export default App;

```
